# Le's Blog

基于 Next.js 开发的个人博客，部署在 Vercel。

## 技术栈

- Next.js 16 + TypeScript
- Tailwind CSS v4
- Framer Motion
- Shiki 代码高亮
- GitHub Api 持久化
- Cloudflare Pages / Vercel 兼容部署

## 本地开发

```bash
pnpm install
pnpm dev
```

访问 http://localhost:2025 即可预览。

## 部署

直接推送 main 分支到 GitHub，Vercel 会自动部署。
